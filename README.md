# Projeto de Testes Manuais - SauceDemo

## Objetivo

Este projeto tem como objetivo demonstrar conhecimentos em testes manuais, criação de casos de teste, documentação de bugs e validação funcional da aplicação SauceDemo.

---

## Aplicação Testada

- SauceDemo
- https://www.saucedemo.com/

---

## Escopo dos Testes

Funcionalidades testadas:
- Login
- Logout
- Carrinho
- Navegação
- Mensagens de erro

---

## Artefatos do Projeto

- Plano de testes
- Casos de teste
- Bug reports
- Checklist
- Evidências

---

## Ferramentas Utilizadas

- GitHub
- Markdown
- Navegador Google Chrome
