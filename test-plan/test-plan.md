
# Plano de Testes - SauceDemo

## Objetivo

Validar as funcionalidades principais da aplicação SauceDemo garantindo que o sistema funcione corretamente para o usuário final.

---

## Escopo

Serão testadas as funcionalidades:
- Login
- Logout
- Adição ao carrinho
- Remoção do carrinho
- Navegação entre páginas

---

## Tipo de Teste

- Testes funcionais
- Testes positivos
- Testes negativos

---

## Ambiente de Teste

- Navegador: Google Chrome
- Sistema Operacional: Windows 11

---

## Critérios de Sucesso

O sistema deverá:
- Permitir login válido
- Bloquear login inválido
- Adicionar produtos ao carrinho
- Remover produtos corretamente

---

## Riscos

- Instabilidade do ambiente
- Lentidão de carregamento
